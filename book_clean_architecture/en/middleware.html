Middleware

Invisible guardians of your application. Code that "embraces" every request.

How It Works (Onion Model)

Imagine Middleware as layers of an onion around your application. A request must pass through all layers to reach the core (Router â†’ Controller â†’ Use Case). The response flies back through the same layers.

Request â¡ [ Middleware 1 ] â¡ [ Middleware 2 ] â¡ [ ROUTER / APP ]
Response â¬… [ Middleware 1 ] â¬… [ Middleware 2 ] â¬… [ RESULT ]

What's It Used For? (Cross-Cutting Concerns)

  • Authentication: Who is this user? (JWT, Session)
  • Context Management: Request ID, DB session, user_id.
  • Logging: Recording all incoming requests and errors.
  • CORS: Allowing access from the frontend.
  • GZIP / Compression: Compressing responses.
  • Rate Limiting: Protection from spam (N requests per minute).
  • Error Handling: Global exception handling (500 Server Error).

Implementation Examples

1. Logging and Timing

Classic example: measure how long a request takes to process.

import time
from starlette.middleware.base import BaseHTTPMiddleware
from fastapi import Request

class TimingMiddleware(BaseHTTPMiddleware):
    async def dispatch(self, request: Request, call_next):
        start_time = time.time()
        
        # ğŸŸ¢ BEFORE processing the request
        print(f"Start: {request.method} {request.url}")
        
        # Passing control further down the chain
        response = await call_next(request)
        
        # ğŸ”´ AFTER processing
        process_time = time.time() - start_time
        response.headers["X-Process-Time"] = str(process_time)
        print(f"Done in {process_time:.4f}s")
        
        return response

2. Context Management (Correlation ID)

To track logs of the same request, it's useful to assign it a unique ID (Request ID) and pass it to the context.

import uuid
from contextvars import ContextVar

# Global variable for Request ID
request_id_ctx: ContextVar[str] = ContextVar("request_id", default=None)

class RequestIDMiddleware(BaseHTTPMiddleware):
    async def dispatch(self, request: Request, call_next):
        # Generate ID
        req_id = str(uuid.uuid4())
        
        # Set in ContextVar
        token = request_id_ctx.set(req_id)
        
        try:
            # Execute request
            response = await call_next(request)
            # Return to client so they can provide this ID to support
            response.headers["X-Request-ID"] = req_id
            return response
        finally:
            # Clear context (important for asyncio!)
            request_id_ctx.reset(token)
