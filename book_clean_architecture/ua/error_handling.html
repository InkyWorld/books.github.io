    """
    –ì–ª–æ–±–∞–ª—å–Ω—ã–π –æ–±—Ä–∞–±–æ—Ç—á–∏–∫ –≤—Å–µ—Ö –ø–æ–º–∏–ª–æ–∫ –¥–æ–º–µ–Ω–∞.

            "detail": str(exc),

    )
